# Sticky Price Inflation Forecasting

## Overview
Developed a time series forecasting model in R to analyze sticky price inflation trends using macroeconomic data.

## Methods
- ARIMA modeling
- Forecasting
- Residual diagnostics
- Time series decomposition

## Tools
- R
- forecast package
- ggplot2

## Results
Generated inflation forecasts and analyzed persistence patterns using historical macroeconomic indicators.
